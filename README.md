# Minimedi

To intiliatize webiste -
    in parent Directory write
    ```make initializeApp```
    then
    ```
    make all
    ```
    or ```make run```
    

And to Run
  ```
  make run
  ```
  
  OR 
  in one Terminal start the server by
  ```
  cd backend
  npm install
  node server.js
  ```
  
  and in another terminal
  ```
  cd frontend
  npm install
  npm run dev
  ```
